# 2-Sprint-Mission

## 프로젝트 설명

이 프로젝트는 코드잇 부트캠프의 첫 번째, 두 번째 스프린트 미션으로, 주어진 API를 이용하여 상품 목록과 게시글 목록을 조회하고 특정 데이터를 처리하여 화면에 표시하는 기능을 구현합니다.

## 사용 기술

* JavaScript
* Node.js
* git
